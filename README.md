# Getting and cleaning data - course project

The project contains a single executable R file and data directory.

Workflow of the file:
1.  Initialization section: reading feature names, subjects, activity names and labels
from both test and main directory. Test data is appended to the main one.
2. Main section
* Replacing 
* Function to return activity name
* Replacing activity ID with activity name
* Creating vector with column names to extract from the dataset
* Extracting necessary columns
* Putting all together with subjects and activity labels
* Applying aggreation, removing unnecessary columns and renaming grouping columns
* Creating the file
